# 📚 Student Registration Program

## 🤓 Description
This project is a simple Python program to register students and their grades. It is a basic system that stores the name of each student and their grades in three subjects: Math, Biology, and Socials. The purpose of the project is to practice basic programming concepts such as lists, loops, dictionaries, and user input in Python.

## 🧑🏻‍💻 How it works
1. The program starts by displaying a welcome message for the user.
2. It asks the user how many students they want to register in the system.
3. For each student, the program asks for the student's name.
4. After that, the program asks for the grades in the subjects Math, Biology, and Socials.
5. The program saves the student's name and grades inside a list using dictionaries.
6. Finally, the program prints all the registered students along with their grades on the screen.

## Status
> The program is currently running in the console and it is collecting and storing student information during execution. It is working as a basic student registration system for practice purposes.
